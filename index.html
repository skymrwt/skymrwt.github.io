Mac终端命令

显示隐藏文件夹(更改true和false)

defaults write com.apple.finder AppleShowAllFiles -boolean true ; killall Finder

在Mac、Linux 终端显示 Git 当前所在分支

进入你的home目录
cd ~
编辑.bashrc文件
vi .bashrc
将下面的代码加入到文件的最后处

export PS1='\u@\h \[\033[01;36m\]\W\[\033[01;32m\]$(git_branch)\[\033[00m\] \$ '
保存退出
执行加载命令
source ./.bashrc
完成
Mac 下面启动的 shell 是 login shell，所以加载的配置文件是.bash_profile，不会加载.bashrc。如果你是 Mac 用户的话，需要再执行下面的命令，这样每次开机后才会自动生效：

echo "[ -r ~/.bashrc ] && source ~/.bashrc" >> .bash_profile

正则及其应用

* 匹配前面一个表达式0次或多次,等价于{0,}
+ 匹配前面一个表达式1次或多次,等价于{1,}
? 匹配前面一个表达式0次或1次,等价于{0,1}

test方法
  • regexObj 为正则表达式
  • str 用来与正则表达式匹配的字符串
  • 返回值 如果正则表达式与指定的字符串匹配 ，返回true；否则false。
exec方法
  • regexObj 为正则表达式
  • str 用来与正则表达式匹配的字符串

  • 返回值 匹配成功,返回一个数组,匹配失败,返回null

    match方法

  • 返回值 匹配成功返回数组,否则返回null

search方法
  • 返回值 返回匹配到的位置索引，或者在失败时返回-1

兼容性问题

在不同的浏览器下,上传文件的事件对象的属性不同

  1. 谷歌浏览器
    e.path[0].files[0]
    可通过URL.createObjectURL(e.path[0].files[0]);将其转成16进制直接当做img标签的src使用

  2. 火狐浏览器
    e.originalTarget.files[0]);
    转换方法URL.createObjectURL()

  3. safari浏览器
    e.target.files[0]);
    转换方法URL.createObjectURL()

  4. 360浏览器7.0版本

    转换方法URL.createObjectURL()
